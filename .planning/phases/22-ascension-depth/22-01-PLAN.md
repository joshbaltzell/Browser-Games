### T04 — Add `showAscensionPrompt()`, `acceptAscension()`, `skipAscension()` to `game.js`

**File:** `games/neon-swarm/game.js`

**Read first:** `openModifierSelection()` and `chooseModifier()` (Phase 06, around lines 1131–1160) for the overlay show/hide pattern; `updateHud()` for where the HUD badge should be updated from.

**Location:** Place the three new functions together, near the other overlay-management functions (`openModifierSelection`, `chooseModifier`, `openSkillTree`, ~line 1641). Add them as a named block with a `// Phase 22 — Ascension Depth` comment header.

**Code to add:**

```js
// ── Phase 22: Ascension Depth ────────────────────────────────────────
function showAscensionPrompt() {
  ascensionPromptActive = true;
  const tier = ascensionNextIdx + 1; // 1, 2, or 3
  const romanMap = ["I", "II", "III"];
  dom.ascension.querySelector("#ascension-title").textContent = "ASCENSION " + romanMap[ascensionNextIdx];
  // Cumulative stats note (shown only when already ascended at least once)
  const cumEl = dom.ascension.querySelector("#ascension-cumulative");
  if (ascensionLevel > 0) {
    const cumHp  = (Math.pow(1.25, ascensionLevel + 1)).toFixed(2);
    const cumSpd = (Math.pow(1.15, ascensionLevel + 1)).toFixed(2);
    const cumXp  = (Math.pow(1.5,  ascensionLevel + 1)).toFixed(2);
    cumEl.textContent = "If accepted — cumulative: ×" + cumHp + " HP  ×" + cumSpd + " speed  ×" + cumXp + " XP";
  } else {
    cumEl.textContent = "";
  }
  dom.ascension.classList.remove("hidden");
}

function acceptAscension() {
  ascensionLevel++;
  ascensionNextIdx++;
  ascensionPromptActive = false;
  dom.ascension.classList.add("hidden");
  // Update HUD badge immediately
  const romanMap = ["I", "II", "III"];
  dom.ascensionBadge.textContent = "[" + romanMap[ascensionLevel - 1] + "]";
  dom.ascensionBadge.classList.remove("hidden");
}

function skipAscension() {
  ascensionNextIdx++;
  ascensionPromptActive = false;
  dom.ascension.classList.add("hidden");
}
// ── end Phase 22 ─────────────────────────────────────────────────────
```

**Button wiring.** In the section where button click listeners are registered (same area as `start-btn` / `restart-btn` / `ascend-btn` wiring, around lines 195–200), add:

```js
dom.ascendBtn.addEventListener("click", acceptAscension);
dom.skipBtn.addEventListener("click", skipAscension);
```

**Acceptance criteria:**
- `showAscensionPrompt()` sets `ascensionPromptActive = true`, updates `#ascension-title` text, populates or clears `#ascension-cumulative`, and shows the overlay.
- `acceptAscension()` increments `ascensionLevel` and `ascensionNextIdx`, clears `ascensionPromptActive`, hides the overlay, and updates the HUD badge.
- `skipAscension()` increments only `ascensionNextIdx`, clears `ascensionPromptActive`, and hides the overlay.
- Both buttons have click listeners registered.
