# Phase 08 — Death Bomb (Last Stand) Upgrade — Plan 08-01

## Goal

Add "Last Stand" as a new upgrade in the `UPGRADES` pool. When the player takes
damage that would reduce HP to 0 or below, Last Stand intercepts the killing blow,
restores HP to 5, consumes a charge, and triggers a dramatic Bomb screen-clear with
gold floater, screen shake, slow-mo, white flash, and a longer invuln window. The
charge mechanic uses an integer `player.lastStandCharges` (0-2) per CONTEXT D-11 —
each pick adds one charge up to a max of 2; each lethal interception decrements by one.

All work happens in `games/neon-swarm/game.js`. No HTML/DOM changes are required.
