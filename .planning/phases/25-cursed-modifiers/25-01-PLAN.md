# Phase 25: Cursed Modifier Stacking — Plan 25-01

## Overview

Extend the Phase 6 modifier system so players can check multiple modifiers before a run instead of picking one. Taking 2 non-Standard modifiers draws 1 random Curse from a 4-entry `CURSES` pool; taking 3 draws 2 Curses. Curses apply on run start and display as red HUD chips. Finishing a run with 2+ active curses writes a MASOCHIST flag to `localStorage` and surfaces a badge on the game-over screen. Implementation touches the modifier overlay (checkbox-style toggle + confirm button), five game-loop integration points (spawn, drop, skill-tree, HUD, end-game), and two CSS additions (`.curse-chip`, `.masochist-badge`).

---

## Source-of-Truth Reconciliation (read before implementing)

CONTEXT.md is the authoritative decision source. Where plan precision and CONTEXT differ, follow CONTEXT.

- **chosenModifiers vs selectedModifier:** `selectedModifier` is kept as a backward-compatible alias (`chosenModifiers[0] || null`) so the HUD modifier-label wiring in `applyAndStart()` (line ~2621) needs only a small extension, not a rewrite.
- **Standard Run stacking:** if the player checks Standard alongside real modifiers, Standard is silently excluded from `chosenModifiers` at confirm-time — its `apply()` is a no-op anyway, but exclusion keeps curse-count math clean.
- **hudBlind scope:** Curse chips render BEFORE the early-return in `updateHud()` so the player always sees which curses they accepted, even under BLIND.

    function applyAndStart() {
      // Apply all chosen modifiers in order.
      chosenModifiers.forEach(m => { if (m) m.apply(player); });

      // Roll curses based on non-Standard modifier count.
      const nonStandard = chosenModifiers.filter(m => m && m.id !== "standard");
      const curseCount = nonStandard.length >= 3 ? 2 : nonStandard.length >= 2 ? 1 : 0;
      if (curseCount > 0) {
        const shuffled = [...CURSES].sort(() => Math.random() - 0.5);
        shuffled.slice(0, curseCount).forEach(c => c.apply());
      }

      gameState = "playing";
      dom.start.classList.add("hidden");
      dom.gameover.classList.add("hidden");
      dom.levelup.classList.add("hidden");
      dom.modifier.classList.add("hidden");
      dom.hud.classList.remove("hidden");
      lastTime = performance.now();

      // HUD modifier label: comma-joined list for multi-select.
      const nonStandardMods = chosenModifiers.filter(m => m && m.id !== "standard");
      if (nonStandardMods.length > 0) {
        dom.modifierLabel.textContent = nonStandardMods.map(m => m.name).join(", ");
        dom.modifierLabel.classList.remove("hidden");
      } else {
