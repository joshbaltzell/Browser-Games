# Phase 14: Kill Streak Milestone System — Plan

## Overview

This plan adds three escalating kill-streak power states (RUSH, FRENZY, RAMPAGE) gated at combo thresholds 10/25/50. All changes are confined to `game.js`. Six targeted edits cover: player initialization, milestone activation inside `killEnemy()`, state revert inside the combo-decay block of `update()`, a RAMPAGE aura in `drawPlayer()`, a state label in the combo HUD block of `render()`, and afterimage color tinting in `executeDash()`. No new arrays, no new globals, no HTML changes required.

### T02 — Milestone activation in `killEnemy()`

**File:** `game.js`
**Location:** `killEnemy()`, ~line 1344. Find the line `combo++;` (or equivalent increment). Immediately after that line, before any other combo-related logic, insert the threshold checks.

**Code to add** (after `combo++`):
```js
// --- Kill-streak milestone activation ---
if (combo >= 50 && player.comboMilestone === 'frenzy') {
    player.comboMilestone = 'rampage';
    player.baseDamage = player.damage;
    player.baseProjSpeed = player.projectileSpeed;
    player.damage *= 1.5;
    player.projectileSpeed *= 1.4;
    floaters.push({ x: player.x, y: player.y - 30, text: 'RAMPAGE!', color: '#ffffff', life: 1.4, vy: -60, size: 22 });
} else if (combo >= 25 && player.comboMilestone === 'rush') {
    player.comboMilestone = 'frenzy';
    player.baseFire = player.fireInterval;
    player.fireInterval *= 0.7;
    floaters.push({ x: player.x, y: player.y - 30, text: 'FRENZY!', color: '#ff8800', life: 1.4, vy: -60, size: 22 });
} else if (combo >= 10 && player.comboMilestone === 'none') {
    player.comboMilestone = 'rush';
    player.baseSpeed = player.speed;
    player.speed *= 1.2;
    floaters.push({ x: player.x, y: player.y - 30, text: 'RUSH!', color: '#00ff88', life: 1.4, vy: -60, size: 22 });
}
```

Note: the three checks are ordered largest-to-smallest so that a single kill crossing multiple thresholds simultaneously (e.g. combo jumps straight to 50) will activate the highest applicable state first. Each gate tests the *current* `comboMilestone`, so only one state activates per kill; the player must step through RUSH then FRENZY to reach RAMPAGE naturally.

### T04 — RAMPAGE aura in `drawPlayer()`

**File:** `game.js`

**Code to add:**
```js
// RAMPAGE white bloom aura
if (player.comboMilestone === 'rampage') {
    const auraPulse = 0.05 + 0.10 * Math.abs(Math.sin(elapsed * 6));
    const auraR = player.radius + 20;
    const grad = ctx.createRadialGradient(player.x, player.y, player.radius * 0.5, player.x, player.y, auraR);
    grad.addColorStop(0, `rgba(255,255,255,${(auraPulse * 0.6).toFixed(3)})`);
    grad.addColorStop(1, `rgba(255,255,255,0)`);
    ctx.save();
    ctx.globalCompositeOperation = 'lighter';
    ctx.beginPath();
    ctx.arc(player.x, player.y, auraR, 0, Math.PI * 2);
    ctx.fillStyle = grad;
    ctx.fill();
    ctx.restore();
}
```
