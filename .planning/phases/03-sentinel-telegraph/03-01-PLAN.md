# Phase 03 — Sentinel Telegraph — Plan 03-01

## Goal

Add a red targeting reticle that animates at the player's current position during the
last 0.8 seconds of each Sentinel's shoot cooldown. The reticle shrinks from a large
ring (36px) to a near-point (8px) at the fire moment, gaining opacity and crispness as
it closes, giving the player a clear, readable dodge window. No game logic changes —
this is purely additive rendering computed from existing `shootCd`, `shootInterval`,
and `shootRange` data on ranged enemies (Sentinels).

Implements VIS-02.
