# Phase 18: Dynamic Background Grid Reactivity — Plan 18-01

## Overview

Add a `gridEffects` array whose entries drive per-line color contributions inside
`drawBackground()`. Three event types are supported: `'pulse'` (expanding ring from
explosions), `'tint'` (global cyan wash during Freeze), and `'player'` (inline red
proximity glow during Death Dance — evaluated live, no array entry). Effects are pushed
at `triggerExplosion()` and `activateFreeze()`, aged in a new `updateGridEffects(dt)`
called from `update()`, and consumed in a refactored `drawBackground()`. No HTML, CSS,
or gameplay changes are required.

    ```js
    function drawBackground() {
      // Background fill.
      ctx.fillStyle = "#05050c";
      ctx.fillRect(0, 0, W, H);

      const grid = 48;
      const ox = (elapsed * 12) % grid;

      // Compute global tint contribution from any active 'tint' effects (e.g. Freeze).
      // tintAlpha is additive on top of base line opacity.
      let tintAlpha = 0;
      let tintR = 0, tintG = 229, tintB = 255; // base cyan matches grid default
      for (const e of gridEffects) {
        if (e.type === 'tint') {
          const a = (1 - e.age / e.maxAge) * 0.7;
          if (a > tintAlpha) {
            tintAlpha = a;
            // Freeze tint: shift toward #78dcff = rgb(120, 220, 255)
            tintR = 120; tintG = 220; tintB = 255;
          }
        }
      }

      // Base line opacity (0.05) boosted by tint.
      const baseAlpha = 0.05 + tintAlpha * 0.17;
      const baseColor = `rgba(${tintR},${tintG},${tintB},${baseAlpha.toFixed(3)})`;

      // Draw each vertical line individually to allow per-line pulse contribution.
      ctx.lineWidth = 1;
      for (let lx = -ox; lx < W; lx += grid) {
        // Accumulate pulse brightness for this line's x-coordinate.
        let extra = 0;
        for (const e of gridEffects) {
          if (e.type === 'pulse' && e.radius > 0) {
            const dist = Math.abs(lx - e.x);
            if (dist <= e.radius) {
              const t = 1 - dist / e.radius;
              extra += t * 0.20;
            }
          }
        }
        extra = Math.min(extra, 0.50); // clamp so lines never blow out
        if (extra > 0) {
          // Pulse lines glow orange-white.
          const pulseA = (baseAlpha + extra).toFixed(3);
          ctx.strokeStyle = `rgba(255,180,80,${pulseA})`;
        } else {
          ctx.strokeStyle = baseColor;
        }
        ctx.beginPath();
        ctx.moveTo(lx, 0);
        ctx.lineTo(lx, H);
        ctx.stroke();
      }

      // Draw each horizontal line individually.
      for (let ly = -ox; ly < H; ly += grid) {
        let extra = 0;
        for (const e of gridEffects) {
          if (e.type === 'pulse' && e.radius > 0) {
            const dist = Math.abs(ly - e.y);
            if (dist <= e.radius) {
              const t = 1 - dist / e.radius;
              extra += t * 0.20;
            }
          }
        }
        extra = Math.min(extra, 0.50);
        if (extra > 0) {
          const pulseA = (baseAlpha + extra).toFixed(3);
          ctx.strokeStyle = `rgba(255,180,80,${pulseA})`;
        } else {
          ctx.strokeStyle = baseColor;
        }
        ctx.beginPath();
        ctx.moveTo(0, ly);
        ctx.lineTo(W, ly);
        ctx.stroke();
      }

      // Death Dance proximity glow: red lines within 150px of player when active.
      const deathDanceActive = player.deathDance && player.hp < player.maxHp * 0.25;
      if (deathDanceActive) {
        const pulse = 0.7 + 0.3 * Math.sin(elapsed * 4);
        const ddAlpha = (0.18 * pulse).toFixed(3);
        ctx.strokeStyle = `rgba(255,59,107,${ddAlpha})`;
        ctx.lineWidth = 1;
        for (let lx = -ox; lx < W; lx += grid) {
          if (Math.abs(lx - player.x) <= 150) {
            ctx.beginPath();
            ctx.moveTo(lx, 0);
            ctx.lineTo(lx, H);
            ctx.stroke();
          }
        }
        for (let ly = -ox; ly < H; ly += grid) {
          if (Math.abs(ly - player.y) <= 150) {
            ctx.beginPath();
            ctx.moveTo(0, ly);
            ctx.lineTo(W, ly);
            ctx.stroke();
          }
        }
      }
    }
    ```

    Implementation notes:
    - The per-line loop replaces the original single `ctx.beginPath()` block. Grid spacing

      color changes (CONTEXT D-17).
    - The tint blends the base line color from baseline cyan toward freeze-cyan
      proportionally to how much time remains on the freeze, then fades out (CONTEXT D-13).
    - The pulse falloff uses `1 - dist/radius` (linear), giving maximum brightness at the
      effect origin and falling to 0 at the ring's edge (CONTEXT D-14).
    - Death Dance check exactly mirrors `updateShooting` line ~910 (CONTEXT D-07, D-15).
    - `elapsed` is the same top-level global used by the original scroll offset — accessible
      without any import (CONTEXT, code_context section).
    - Performance: ~32 lines * max 20 effects = ~640 comparisons per frame — negligible
      (CONTEXT D-16).
