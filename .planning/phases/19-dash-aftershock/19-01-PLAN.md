# Phase 19: Dash Aftershock Wave — Plan

### T01 — Add color-field support to `drawBlasts()`

**File:** `games/neon-swarm/game.js`

**Location:** `drawBlasts()`, approximately line 2111. The current line reads:

```js
const color = s.crit ? COLORS.gold : "#ff9f43";
```

**Change:** Replace that single line with:

```js
const color = s.color ?? (s.crit ? COLORS.gold : "#ff9f43");
```

The nullish coalescing operator (`??`) means: if the blast entry has an explicit `color` field (truthy string), use it; otherwise fall through to the existing crit/normal logic. Entries created by `triggerExplosion()` (~line 1189), `updateBullets()` (~line 1275), `killEnemy()` (~line 1367), and all other existing callers do not include a `color` property, so `s.color` will be `undefined` and the fallback fires — behavior is byte-for-byte identical for them. The new dash blasts will include `color: '#00e5ff'` and will therefore render cyan.

**No other changes to `drawBlasts()`.**

### T04 — Play landing click sound in `executeDash()`

**File:** `games/neon-swarm/game.js`

**Location:** `executeDash()`, immediately after the Phase Runner spark block added in T03 (still inside `executeDash()`, before the closing brace).

Add:

```js
// Landing click sound (Phase 19)
if (!muted && audioCtx) {
  const osc = audioCtx.createOscillator();
  const gain = audioCtx.createGain();
  osc.type = 'sine';
  osc.frequency.value = 600;
  gain.gain.value = 0.15;
  osc.connect(gain);
  gain.connect(audioCtx.destination);
  osc.start(audioCtx.currentTime);
  osc.stop(audioCtx.currentTime + 0.06);
  osc.onended = () => gain.disconnect();
}
```

`muted` is a global boolean (~line 273 area, reset in `initGame()`). `audioCtx` is a global variable initialized in Phase 1 (~line 273 area). Both are already guarded in this exact pattern throughout the game's audio functions. The `onended` callback disconnects the gain node to avoid the Web Audio node accumulation memory leak. Duration is 0.06 s and gain is 0.15 — quiet enough not to compete with combat sounds but audible on a landing.
