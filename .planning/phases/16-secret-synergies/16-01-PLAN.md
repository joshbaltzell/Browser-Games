# Phase 16: Secret Synergies — Plan

## Overview

Add six hidden cross-skill synergies that fire a one-time discovery floater and activate a
lasting effect the instant a player completes the required skill combination inside `buySkill()`
(~line 1754). A new `player.triggeredSynergies` Set tracks which have fired this run. Effects
reuse `triggerSlowmo`, `effectiveFireInterval`, `player.orbitals`, `player.invuln`, and the
existing lifesteal calculation — no new rendering systems needed. A SYNERGY CODEX section is
appended to the skill tree panel in `renderSkillTree()`, showing discovered names in gold and
`???` for the rest.

All decisions are from `.planning/phases/16-secret-synergies/16-CONTEXT.md`.
