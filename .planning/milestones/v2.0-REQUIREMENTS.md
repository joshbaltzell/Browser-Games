## v1 Requirements — Final Status

### Audio

- [x] **AUDIO-01**: Game plays synthesized tones for all significant game events (shoot, hit, kill, level-up, bomb, freeze, overdrive activate)  
  **Outcome:** Validated — Phase 1. AudioContext with per-call OscillatorNode synthesis. sndLevelUp() wired at audit (was defined but not called). All 8 events have sound.

### Visual Clarity

- [x] **VIS-01**: Each enemy type renders with a distinct geometric silhouette — darters as triangles, brutes as hexagons, sentinels as diamonds, spores as pentagons; chasers remain circles  
  **Outcome:** Validated — Phase 2. Canvas 2D path drawing per type. Glow and HP bars preserved.

- [x] **VIS-02**: Sentinels display a targeting reticle at the player's current position that shrinks to a point in the 0.8 seconds before firing  
  **Outcome:** Validated — Phase 3. drawSentinelTelegraphs() reads shootCd per Sentinel. Real-time tracking. Freeze-suppressed.

### UX / Information

- [x] **UX-01**: Active power-up duration (Freeze and Overdrive) is displayed as a text countdown with a depleting bar  
  **Outcome:** Validated — Phase 4. drawPowerupTimers() shows both bars simultaneously when active.

### Pacing

- [x] **PACE-01**: After 60 seconds of play, the game periodically announces specific enemy surge events 3 seconds before the burst spawns  
  **Outcome:** Validated — Phase 5. 4 surge types with 30–50s cooldown. Color-coded warnings. Normal spawning continues during warning.

### Meta / Replayability

- [x] **META-01**: Before each run, player chooses one of four run modifiers: Glass Cannon (2× damage, 50% HP), Headstart (start Level 5 with 3 upgrades), Bullet Hell (enemy ranged fire 3×, XP drops 2×), or No Modifier  
  **Outcome:** Validated — Phase 6. 4-card modal UI, keyboard shortcuts 1–4, persistent HUD label.

### Upgrades / Mechanics

- [x] **UPGR-01**: Chain Lightning upgrade — on bullet impact, arcs to nearest other enemy within 150px for 55% of shot damage; stackable  
  **Outcome:** Validated — Phase 7. Fusion skill (splitshot + explosives). chainCount stackable. Cyan polyline arc visual.

- [x] **UPGR-02**: Last Stand (Death Bomb) upgrade — once per run, survive lethal damage with 5 HP and trigger a Bomb  
  **Outcome:** Validated — Phase 8. Fusion skill (regen + overcollect). 2-charge max. Both contact and projectile damage paths covered.

### Controls

- [x] **CTRL-01**: Pressing Shift dashes the player 120px in their current movement direction, granting 0.35s invuln frames, with a 1.5s cooldown tracked by a HUD indicator  
  **Outcome:** Validated — Phase 9. executeDash() with afterimages and #dash-ready HUD indicator.

### Flair / Identity

- [x] **FLAIR-01**: Game detects recognized upgrade combinations after each pick and flashes a build title; current build name persists in small HUD text  
  **Outcome:** Validated — Phase 10. 18+ named builds. checkBuildName() called after every skill buy and at Headstart start (fixed at audit). Gold floater + HUD persistence.
